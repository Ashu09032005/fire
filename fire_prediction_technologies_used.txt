
Technologies and Libraries Used in the Fire Prediction and Spread Modeling Project
----------------------------------------------------------------------------------

1. Programming Language:
   - Python

2. Data Handling and Processing:
   - pandas: Data loading, manipulation, and preprocessing
   - numpy: Numerical computations

3. Data Visualization:
   - matplotlib.pyplot: Plotting bar plots and other charts
   - seaborn: Correlation heatmap and pair plots

4. Machine Learning (ML) Models and Tools:
   - scikit-learn (sklearn):
     - Model training and evaluation: RandomForestClassifier, DecisionTreeClassifier, AdaBoostClassifier
     - Regression: RandomForestRegressor for spread distance prediction
     - Data scaling: StandardScaler
     - Train/test splitting: train_test_split
     - Evaluation metrics: accuracy_score, precision_score, recall_score, f1_score, roc_auc_score

   - xgboost:
     - XGBClassifier: Gradient boosting model for classification

5. Geospatial Visualization:
   - folium: Creating interactive maps to visualize fire origin and predicted spread areas
   - folium.plugins.MeasureControl: Map measurement tool
   - geopy.distance: (Imported for geographical calculations, although not actively used in current code)

6. Output Formats:
   - HTML file: fire_spread_map.html - for visualizing the predicted spread on a map

----------------------------------------------------------------------------------
Note: The project combines classification (Fire/No Fire) and regression (spread prediction in km) tasks,
along with data visualization and geospatial mapping for comprehensive wildfire modeling.
